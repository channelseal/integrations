# ChannelSeal Integrations

> **⚠️ Early Stage**: Integrations are currently in early development. Features and APIs are subject to change.

## Overview

ChannelSeal Integrations describes various integrations with the ChannelSeal platform. It provides detailed instructions on how you can integrate various applications and enterprise systems with the ChannelSeal platform. 

Note: This is an experimental feature being actively developed.

## Current Status

**Alpha** - Limited functionality, breaking changes expected

## Integrations

* [OpenTelemetry](./otlp-collector/README.md) export your HTTP API specific OpenTelemetry signals to ChannelSeal
* [Squid-OpenTelemetry](./squid-otel/README.md) export logs of HTTP API calls from [Squid](https://www.squid-cache.org/) forward proxy to ChannelSeal via OpenTelemetry Collector

## Getting Started

### Prerequisites
- ChannelSeal account
- API access enabled
